Qual é a diferença entre linguagem de programação server-side e client-side?

A diferença está principalmente em onde o código vai ser executado, a programação server-side que é back-end faz o
computador guardar e entender as informações, e a programação client-side é o front-end, que não seria bem programação
pois é feito com linguagem de marcação por exemplo, HTML, CSS e JavaScript.
